# Clinical NER Annotation Interface & Deep Learning Backend

A Deep Learning Lab project for Named Entity Recognition (NER) in English doctors' letters and clinical case reports.

## Team Members

- Aveen Vas 
- Byna Rithika
- Dhanvarshini Ramesh

## Project Overview

This project aims to build an end-to-end system for clinical Named Entity Recognition (NER), combining:

- A deep learning backend for automatic entity detection
- An interactive web-based annotation interface
- Human-in-the-loop correction and annotation workflow
